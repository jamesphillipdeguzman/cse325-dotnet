# CSE325 .NET Project – ContosoPizza & Sales Summary

## Overview

This repository contains two main exercises:

### 1. ContosoPizza Web API

A REST API built with ASP.NET Core that manages pizza records. It supports:

- Create (POST)
- Read (GET)
- Update (PUT)
- Delete (DELETE)

### 2. Sales Summary Console App

A C# application that:

- Reads sales data from multiple store folders
- Parses JSON sales files
- Calculates total sales per store
- Computes overall sales total
- Outputs results to `totals.txt`

## Tech Stack

- C#
- ASP.NET Core Web API
- .NET Console Application
- Newtonsoft.Json
- File I/O
